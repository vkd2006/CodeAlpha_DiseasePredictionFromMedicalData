# Disease Prediction from Medical Data 🏥

## Objective
Predict the possibility of Diabetes based on patient medical data.

## Dataset
- Pima Indians Diabetes Dataset (UCI ML Repository)
- 768 patient records
- 8 medical features

## Features Used
Pregnancies, Glucose, Blood Pressure, Skin Thickness, 
Insulin, BMI, Diabetes Pedigree, Age

## Algorithms Used
| Model | Accuracy |
|---|---|
| Logistic Regression | 75.32% |
| Random Forest | 74.03% |
| SVM | 73.38% |
| XGBoost | add yours |

## Libraries Used
- Python
- Scikit-learn
